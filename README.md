# Sqairz Dashboard

Fastest deployment path:

## Local run
```bash
npm install
npm run dev
```

## Deploy to Vercel
- Create a new Vercel project
- Upload this folder or connect a repo
- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`

## Deploy to Netlify
- Import the project
- Build command: `npm run build`
- Publish directory: `dist`
